How it Works

三步完成全流程

Three steps from meeting to structured notes — all automated, all local

01
Record
一键录制
点击开始录制，AIMA 同步采集麦克风和系统音频。支持随时暂停和恢复，不影响后续处理质量。
02
Transcribe + Diarize
自动转写 + 说话人识别
Whisper large-v3-turbo 本地转写，支持中文和英文。pyannote 进一步分离说话人，生成带角色的逐字稿。
03
Summarize
生成结构化纪要
Gemma 4 本地大模型根据转录内容输出七节结构化会议纪要，支持随时重新生成。
System Requirements

系统要求

What you need to run AIMA

组件要求
芯片Apple Silicon（M1 / M2 / M3 / M4 系列）
系统macOS 14 Sonoma 及以上
内存≥ 8GB（推荐 ≥ 16GB；内存越大，可使用的 AI 模型越强）
存储3–22GB 模型缓存空间（根据内存档位自动选择模型）

⚠️ 暂不支持 Intel Mac。AIMA 依赖 Apple MLX 框架，该框架仅支持 Apple Silicon。

Get Started

三步开始使用

Download, install, and follow the in-app guided setup

1
下载并安装
从 GitHub Releases 下载最新版 AIMA-x.x.x.zip， 解压后将 AIMA.app 拖入 /Applications。 首次打开时右键 → 打开，绕过 Gatekeeper 提示。
2
按引导安装依赖
App 内置环境检测页面，会自动检测 Homebrew、Python、ffmpeg、Whisper、Gemma 等依赖是否就绪， 并提供一键安装命令。首次配置约需 10–30 分钟（主要是模型下载时间）。
3
开始录制
依赖就绪后即可录制。会议结束后 AIMA 自动处理录音并在历史列表中展示结构化纪要。 所有内容保存在本地，随时可查阅或重新生成。
